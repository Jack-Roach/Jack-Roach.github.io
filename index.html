Classifying Reddit Posts by Title Using NLP

The goal of this project was to choose two subreddits and train a binary classification model to correctly predict which subreddit an arbitrary post originated from using natural langage processing. I scraped 5,000 thousand posts from each subreddit using Python's Request library and the Pushshift Reddit API. The model is trained off of the titles of each sample post.

Forecasting Next-day Rain Using AU Weather Data

In this timed project for a General Assembly hackathon, my objective was to find a usable dataset online and make a predictive model for it with a five hour time constraint. I used 10-year Australian weather data to make a binary classification model that determined whether it was going to rain the following day based on a given date's weather features.

Analyzing Writing Styles in the Harry Potter Series Using NLP

I used natural language processing to analyze the writing styles of the seven original Harry Potter novels in order to determine if there was a difference in writing styles between books. Upon constructing a doc2vec model for the corpora, I attempted to make a model with the goal of correctly identifying an arbitrarily selected paragraph's book of origin based on its writing style.
